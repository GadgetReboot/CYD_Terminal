SerialOutputData is a test sketch for a separate ESP32 to send text and data serially so it can be used as a text/data input to test the CYD Serial Terminal/Plotter.<br><br>
Connect Tx out of the ESP32 running this sketch to Rx in on the CYD and it will send a few hundred numbered lines of text, followed by a repeating pattern of random numbers and sine waves to test the plotter on the CYD.
